# Приглашение на свадьбу — Павел & Мария

> Одностраничный сайт-приглашение с анкетой для гостей.

Сайт свадебного приглашения. Гости открывают анимированный конверт, знакомятся с информацией (место, дресс-код, детали, FAQ) и заполняют анкету. Поддержка русского языка, фоновой музыки и адаптивной вёрстки.

## Быстрый старт

```bash
npm ci
cp .env.example .env
# Заполните .env (TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID, ADMIN_USER, ADMIN_PASSWORD)
npm start
```

Сайт: http://localhost:3000 · Админ-панель: http://localhost:3000/admin

## Основные возможности

- **Конверт** — анимированное открытие приглашения
- **Музыка** — фоновый плеер
- **Анкета гостей** — ФИО, транспорт, дедлайн 15.04.2026
- **Сохранение данных** — анкета привязана к анонимному ID в localStorage
- **Админ-панель** — таблица заявок, экспорт CSV/Excel, Basic Auth
- **Docker** — готов к деплою через `docker compose up -d`

## Пример

```bash
# Локальная разработка (hot reload)
npm run dev

# Через Docker
docker compose up -d
```
