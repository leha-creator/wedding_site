# Начало работы

## Требования

- **Node.js** 20+
- **npm** (идёт в комплекте с Node.js)

Для деплоя через Docker — Docker и Docker Compose.

## Установка

```bash
git clone <repo-url>
cd pavel-maria
npm ci
```

## Настройка

1. Создайте `.env` из примера:

   ```bash
   cp .env.example .env
   ```

2. Заполните переменные (см. [Конфигурация](configuration.md)):

   | Переменная | Обязательность | Описание |
   |------------|----------------|----------|
   | `PORT` | нет (3000) | Порт сервера |
   | `DB_PATH` | нет | Путь к SQLite |
   | `ADMIN_USER`, `ADMIN_PASSWORD` | да (для /admin) | Логин админки |
   | `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID` | нет | Уведомления в Telegram |

## Первый запуск

```bash
npm start
```

Откройте http://localhost:3000 — главная страница.  
Админ-панель: http://localhost:3000/admin (потребуется Basic Auth).

## Режим разработки

```bash
npm run dev
```

Сервер перезапускается при изменении `server.js` и файлов в `src/`.

## Проверка

- Главная страница открывается
- Конверт открывается по клику
- Анкета «заполнить анкету» открывает модал
- После отправки — данные сохраняются (если настроены TELEGRAM, придёт уведомление)
